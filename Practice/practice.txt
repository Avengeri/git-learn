Задача: Создать систему управления университетом. Университет состоит из факультетов, на которых обучаются студенты. Каждый студент имеет имя, возраст и список оценок по предметам. Также у студента есть факультет, на котором он учится.

1.  Создайте структуру `Subject` для представления предмета с его названием и оценкой.
2.  Создайте структуру `Student` для представления студента с его именем, возрастом, списком оценок и факультетом.
3.  Создайте структуру `Faculty` для представления факультета с его названием и списком студентов.
4.  Реализуйте функцию, которая будет вычислять средний балл студента.
5.  Реализуйте функцию, которая будет вычислять средний балл по предмету для всего факультета.
6.  Используйте указатели для изменения данных студента и его оценок.
