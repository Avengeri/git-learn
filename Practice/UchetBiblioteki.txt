Задача: Создать систему учета и управления библиотекой.
В библиотеке хранятся книги разных жанров. У каждой книги есть название, автор и жанр.
Также в библиотеке есть читатели, которые могут брать книги на чтение.
У читателя есть имя, возраст и список взятых книг.

Создайте структуру Book для представления книги с полями Title, Author и Genre.
Создайте структуру Reader для представления читателя с полями Name, Age и BorrowedBooks (список взятых книг).
Создайте интерфейс LibraryItem с методом DisplayInfo() string, который будет возвращать строку с информацией о книге или читателе.
Реализуйте метод DisplayInfo() для структуры Book, который будет возвращать информацию о книге в формате "Название: <название>, Автор: <автор>, Жанр: <жанр>".
Реализуйте метод DisplayInfo() для структуры Reader, который будет возвращать информацию о читателе в формате "Имя: <имя>, Возраст: <возраст>".
Создайте метод BorrowBook(book *Book) для структуры Reader, который будет добавлять книгу в список взятых книг читателя.
В функции main, создайте несколько книг и несколько читателей. Далее, выводите информацию о них с помощью метода DisplayInfo().
Эта задача позволит вам практиковать работу со структурами, методами и интерфейсами, а также создание взаимодействия между ними.
