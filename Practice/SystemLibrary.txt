Вам необходимо создать систему управления библиотекой, используя структуры, методы и интерфейсы.
В библиотеке есть книги разных типов (художественная, научная, детская и т.д.).
Каждая книга имеет заголовок, автора и год выпуска. У книг также может быть специфическая информация
(например, ISBN для научных книг или возрастное ограничение для детских книг).

Создайте структуру Book, которая будет представлять книгу с полями: заголовок, автор, год выпуска.
Определите интерфейс LibraryItem, который будет иметь метод Display(), выводящий информацию о книге.
Создайте структуры для разных типов книг (например, FictionBook, ScientificBook, ChildrenBook), которые будут реализовывать интерфейс LibraryItem и также иметь свои дополнительные поля.
Реализуйте метод Display() для каждой из структур, чтобы выводить информацию о каждой книге в нужном формате.
Создайте структуру Library, которая будет представлять библиотеку. Она должна содержать список книг и методы для добавления, удаления и отображения книг.
Реализуйте методы для добавления, удаления и отображения книг в структуре Library.
Создайте функцию Main(), где вы будете создавать несколько экземпляров разных книг, добавлять их в библиотеку, а затем выводить информацию о книгах в библиотеке.
Эта задача позволит вам применить знания о структурах, методах и интерфейсах для создания сложной системы управления библиотекой, где разные типы книг могут быть добавлены, удалены и отображены с использованием общего интерфейса LibraryItem.
